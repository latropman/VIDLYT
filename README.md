# Latropman's VIDLYT (V003 / 2026-03-24) 
• VIDLYT ("видлит", "видлут", "видлайт" — как вам захочется) — скрипт для скачивания видео из большинства соцсетей.

# Начало работы:
• После запуска VIDLYT SETUP V003.bat начнётся автоматическая распаковка файлов в папку C:\Scripts\VIDLYT. Основной функционал завязан на yt-dlp, но опциональна также установка ffmpeg для конвертации видео в MP3, WAV и MP4 с настройкой битрейта. После установки следуйте инструкции: → Копируете ссылку на видео, например, с Youtube → Внутри нужной вам папки нажимаете правой кнопкой мыши → Выбираете Latropman's VIDLYT → Выбираете разрешение: достаточно нажать цифру на клавиатуре, соответствующую нужному формату, а при отсутствии выбора в течение 15 секунд видео загрузится в самом высоком разрешении из доступных → Скачивание начнётся автоматически в ту папку, в которой был запущен процесс скрипта.

# Основные доступные платформы:
• Youtube, Instagram, Twitter, TikTok, Facebook, Vimeo, Rutube, Одноклассники, VK Video. С полным списком можно ознакомиться по ссылке https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md или через комбинацию WIN+R → cmd → yt-dlp --list-extractors.

# Изменения третьей версии
• Установка: Исправлено окно выбора установки ffmpeg-конвертера (ранее при нажатии на "No" конвертер всё равно устанавливался);

• Скачивание (через yt-dlp.exe): Оптимизирован запуск. Удалены лишние проверки, что ускорило процесс подготовки в 4 раза, а также скорость самой загрузки медиафайлов. Добавлены варианты разрешений скачивания: от 144р до best video, а также можно отдельно скачать best audio. Добавлена поддержка скачивания плейлистов в выбранном разрешении: если выберете 360р, все видео будут скачаны в 360р. Добавлена поддержка параллельных загрузок: они будут запущены в разных процессах и не должны создавать конфликтов между собой. Добавлена очистка названий: по окончанию загрузки название файла очищается от эмодзи и других спецсимволов, чтобы у какого-либо софта не было проблем с их импортом, как это бывает, например, в Davinci Resolve. Весь процесс скачивания не выходит за рамки окна cmd, а визуально этапы очищены от "мусорных" технических строк, то есть выводится только нужная информация по скачиваемому контенту.

• Конвертирование (через ffmpeg.exe): Без изменений. Всё также имеется возможность конвертирования в WAV, MP3 и MP4 через контекстное меню, но при условии, что был выбран FFmpeg Converter во время основной установки скрипта.

# Примечание по установке
• В ходе установки скачивание yt-dlp не должно занимать больше 20 секунд (+-5 в завимости от вашего интернет-соединения), а ffmpeg — не больше 1 минуты при условии, что он у вас не установлен (если установлен, путь к нему будет проверен в переменных средах). Если какой-то из этапов сильно затягивается, это значит, что имеются проблемы со стороны серверов, раздающих файлы. В этом случае закройте окно, удалите открывшуюся папку VIDLYT и повторите процесс установки.

# Комментарий автора
• Преимущество данного софта — в его "прозрачности": плагины yt-dlp и ffmpeg изначально не заложены в установочник и скачиваются отдельно из официальных источников свежими версиями, а сам скрипт можно проверить как на GitHub (https://github.com/latropman/VIDLYT), так и банально через блокнот. Скрипт полностью переведён на английский язык, чтобы избежать возможных ошибок, связанных с кодировкой.
