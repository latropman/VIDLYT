# Latropman's VIDLYT (V003 / 2026-03-24) 
• VIDLYT ("видлит", "видлут", "видлайт" — как вам захочется) — скрипт для скачивания видео из большинства соцсетей. Достаточно скопировать ссылку, затем в нужной папке нажать правой кнопкой мыши по свободному пространству и запустить Latropman's VIDLYT.

# Начало работы:
После запуска VIDLYT SETUP V003.bat начнётся автоатическая распаковка файлов в папку C:\Scripts\VIDLYT. Основной функционал завязан на yt-dlp, но опциональна также установка ffmpeg для конвертации видео в MP3, WAV и MP4 с настройкой битрейта.

# Основные доступные платформы:
Youtube, Instagram, Twitter, TikTok, Facebook, Vimeo, Rutube, Одноклассники, VK Video. С полным списком можно ознакомиться по ссылке https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md или через комбинацию WIN+R → cmd → yt-dlp --list-extractors.

# Изменения третьей версии
Скачивание (через yt-dlp.exe): Оптимизирована скорость процесса, а также скорость самой загрузки медиафайлов. Исправлено окно выбора установки ffmpeg-конвертера (ранее при нажатии на "No" конвертер всё равно устанавливался). Весь процесс скачивания не выходит за рамки окна cmd, а визуально этапы очищены от "мусорных" строк, то есть выводится только нужная информация по скачиваемому контенту. Добавлены варианты разрешений скачивания: от 144р до best video, а также можно отдельно скачать best audio. По окончанию загрузки название файла очищается от эмодзи и других спецсимволов, чтобы, например, у монтажного софта не было проблем с их импортом, как это бывает в Davinci Resolve. Можно запускать несколько загрузок параллельно: они будут запущены в разных процессах и не должны создавать конфликтов.

Конвертирование (через ffmpeg.exe): Без изменений. Всё также имеется возможность конвертирования в WAV, MP3 и MP4 через контекстное меню при условии, что был выбран FFmpeg Converter при основной установке скрипта.

# Примечание по установке
В ходе установки скачивание yt-dlp не должно занимать больше 20 секунд (+-5 в завимости от вашего интернет-соединения), а ffmpeg — не больше 1 минуты при условии, что он у вас не установлен (если установлен, путь к нему будет проверен в переменных средах). Если какой-то из этапов сильно затягивается, это значит, что имеются проблемы со стороны серверов, раздающих файлы. В этом случае закройте окно, удалите открывшуюся папку VIDLYT и повторите процесс установки.

# Комментарий автора
Преимущество данного софта — в его "прозрачности": плагины yt-dlp и ffmpeg изначально не заложены в установочник и скачиваются отдельно из официальных источников свежими версиями, а сам скрипт можно проверить как на GitHub (https://github.com/latropman/VIDLYT), так и банально через блокнот.
